## Overview

This flashcards app is a study utility designed for students or anyone looking to learn. Featuring a sleek, responsive UI and a robust modular architecture, this tool allows you to master any subject with ease. The app supports dynamic resizing, session persistence via JSON, and a variety of study modes to optimize your learning workflow.

## Key Features

- **Responsive UI**: A fully dynamic layout that scales fluidly with window resizing, featuring high-visibility HUD elements and premium aesthetics.
- **Session Persistence**: Automatically saves and loads your progress. Resume exactly where you left off, preserving both your studied cards and specific settings.
- **Comprehensive HUD**: Real-time tracking of card progress, mastery counts, and study status messages.
- **Customizable Study Modes**: Toggle between Term or Definition first, and enable/disable shuffling on the fly for new rounds.

## Installation & Setup

1. Clone the repository with `git clone https://github.com/BryanWieschenberg/QuizWiz.git`, enter the directory with `cd QuizWiz`
2. Put your terms and definitions in `cards.csv`
3. Run the main script with `python3 main.py`
